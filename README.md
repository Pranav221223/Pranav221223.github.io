# 🌐 Pranav C Jadhav – Developer Portfolio

A modern **personal portfolio website** showcasing my work in **Full Stack Development and Artificial Intelligence**.  
The site highlights my projects, technical skills, GitHub activity, and provides a contact interface for professional opportunities.

Built with **Tailwind CSS, Three.js, Chart.js, and JavaScript** to create an interactive and visually engaging developer profile.

---

# 🚀 Live Features

• Dark / Light Theme Toggle  
• Animated Three.js particle background  
• Interactive skills chart (Chart.js)  
• Project showcase with sliders  
• GitHub repository dashboard (GitHub API)  
• Project case-study modal system  
• Responsive mobile bottom navigation  
• Animated typing hero section  
• Contact form with success animation
